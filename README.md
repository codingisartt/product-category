# Product Catalog Application

A full-stack Product Catalog application built with:

* Spring Boot (Backend)
* MySQL (Database)
* React (Vite) (Frontend)
* Bootstrap (UI)

---

## Features

* Product listing
* Category-based structure
* Product filtering
* REST API integration

---

## Backend

### Tech Stack

* Java 21
* Spring Boot
* Spring Data JPA
* MySQL

### Features

* Product and Category APIs
* Data seeding
* CORS configuration

---

## Frontend

### Tech Stack

* React (Vite)
* Bootstrap

### Features

* Fetch data from backend
* Display products in card layout
* Responsive UI

---

## Getting Started

### Backend

```bash
cd backend
mvn spring-boot:run
```

Runs on:

```
http://localhost:8080
```

---

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Runs on:

```
http://localhost:5173
```

---

## API Endpoints

### Products

```
GET /api/products
```

### Categories

```
GET /api/categories
```
